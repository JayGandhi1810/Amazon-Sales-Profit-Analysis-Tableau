# 📊 Amazon Sales and Profit Analysis using Tableau

## 📌 Project Overview

This project presents an interactive Tableau dashboard developed to analyze Amazon sales performance between **2011 and 2014**. The dashboard provides business insights into sales, profit, quantity sold, profit margin, product categories, and regional performance through dynamic visualizations and filters.

The objective of this project is to transform raw sales data into meaningful business intelligence that supports data-driven decision-making.

---

## 🎯 Objectives

- Analyze yearly sales performance.
- Evaluate profit across product categories.
- Compare product sales with generated profit.
- Identify high-performing and low-performing states.
- Monitor yearly KPIs.
- Provide an interactive dashboard using Tableau filters.

# 📂 Dataset Information

The dataset contains Amazon sales transactions from **2011 to 2014** and includes various business attributes used for analysis.

### Dataset Features

- Order Date
- Ship Date
- Customer Details
- Product Category
- Product Name
- Sales
- Profit
- Quantity
- Region
- State
- Segment
- Shipping Mode

The dataset was cleaned and analyzed using Tableau to create interactive dashboards for business intelligence reporting.

---

# 📈 Dashboard KPIs

The dashboard monitors the following Key Performance Indicators (KP2011–2014):

- 💰 Total Sales
- 📦 Total Quantity Sold
- 💵 Total Profit
- 📊 Profit Margin (%)

The dashboard also provides:

- Year-wise Sales Analysis
- State-wise Profit Analysis
- Category-wise Profit Distribution
- Product-wise Sales vs Profit Comparison
- Interactive Filters for Detailed Analysis
